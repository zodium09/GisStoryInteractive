# GeoStory Atlas

หนังสือเรียนภูมิศาสตร์กายภาพและธรณีศาสตร์แบบอินเทอร์แอ็กทีฟสำหรับผู้อ่านตั้งแต่ระดับเริ่มต้นถึงผู้เชี่ยวชาญ เนื้อหาเชื่อมแผนที่ ภาพถ่าย แผนภาพกระบวนการ และกรณีศึกษาจากสถานที่จริงทั่วโลก

## ประสบการณ์เรียนรู้

- แผนผังระบบโลก เชื่อมธรณีภาค บรรยากาศ อุทกภาค และชีวภาค
- 8 สาขาวิชา: ธรณีวิทยา ธรณีสัณฐานวิทยา อุทกวิทยา สมุทรศาสตร์ ภูมิอากาศวิทยา นิเวศวิทยา ภูมิศาสตร์มนุษย์ และภัยธรรมชาติ
- 298 บทเรียนพร้อมภาพ แผนที่ดาวเทียม และอินโฟกราฟิกเชิงกระบวนการ
- บทเรียนฉบับสมบูรณ์: ความหมาย ประเภท กระบวนการเกิด หลักฐาน กรณีศึกษา พิกัด และคำถามทบทวน
- เส้นทางเรียนแบบมีลำดับ พร้อมการสำรวจอย่างอิสระ
- ค้นหาแบบ Command Palette ด้วย `Ctrl+K` หรือค้นในคลังด้วย `/`
- บันทึกบทเรียน Dark mode และ reading progress โดยเก็บสถานะในเบราว์เซอร์
- รองรับ responsive, keyboard navigation, reduced motion และแนวทาง WCAG 2.2 AA

## การเปิดใช้งาน

```powershell
node dev-server.mjs
```

จากนั้นเปิด `http://localhost:8080`

แอปเป็น static HTML/CSS/JavaScript และใช้ Leaflet สำหรับแผนที่ ภาพจากภายนอกจึงต้องเชื่อมต่ออินเทอร์เน็ต

## โครงสร้างสำคัญ

- `index.html` โครงหน้าและ semantic landmarks
- `styles.css` ระบบรูปแบบเดิมของคลังบทเรียนและอินโฟกราฟิก
- `v2.css` visual system และ responsive experience รุ่นใหม่
- `app.js` ข้อมูล 298 บทเรียน โครงบทแบบหนังสือ ตัวกรอง แผนที่ reader และ interactions
- `PRODUCT.md` เป้าหมาย ผู้ใช้ และหลักการออกแบบผลิตภัณฑ์
- `DESIGN.md` design tokens และกฎของ visual system
- `assets/ai/earth-system-hero-v2.png` ภาพระบบโลกที่สร้างเฉพาะสำหรับหน้าเปิดเรื่อง
- `assets/lessons/*.webp` คลังภาพ Imagen เชิงวิทยาศาสตร์ 18 ชุดสำหรับจับคู่กับกระบวนการของทุกบทเรียน
