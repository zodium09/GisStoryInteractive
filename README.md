# ภูมิศาสตร์โลก: หนังสือดิจิทัล

หนังสือเรียนภูมิศาสตร์โลกภาษาไทยสำหรับผู้อ่านทั่วไป เรียบเรียงเนื้อหาจากกลไกและกระบวนการไปสู่รูปแบบเชิงพื้นที่ พร้อมภาพเต็มหน้า แผนภาพ ตาราง และสารบัญติดหน้าจอ

## เนื้อหา 6 หมวด

1. ธรณีสัณฐาน
2. ภูมิอากาศวิทยา
3. สมุทรศาสตร์
4. ภูมิภาคและลักษณะกายภาพโลก
5. การกระจายทรัพยากรธรรมชาติ
6. ภูมิศาสตร์อุตสาหกรรม

แต่ละบทมีภาพเปิดบท เนื้อหาแบบบทความ และหัวข้อย่อยฉบับสมบูรณ์รวม 96 เรื่อง ทุกหัวข้อเปิดเป็นหน้าบทเรียนแยกพร้อมภาพที่เลือกตามเนื้อหา คำอธิบายภาพรวม ลำดับกระบวนการ หลักฐาน ตัวอย่างเชิงพื้นที่ ความสำคัญ และสรุปใจความ มีภาพจำลองกระบวนการแบบเคลื่อนไหว แผนภาพและข้อมูลประกอบ รวมถึงการนำทางไปเรื่องก่อนหน้าหรือเรื่องถัดไป รองรับเดสก์ท็อป แท็บเล็ต โทรศัพท์ โหมดกลางวันและกลางคืน

## เปิดใช้งาน

```powershell
node dev-server.mjs
```

จากนั้นเปิด `http://localhost:8080`

## ไฟล์สำคัญ

- `index.html` โครงหน้าหนังสือและส่วนแสดงผล
- `app.js` เนื้อหาทั้งหกหมวดและระบบเปลี่ยนบท
- `subtopic-content.js` รายละเอียดหัวข้อย่อยทั้ง 96 เรื่อง
- `topic.html`, `topic-page.js`, `topic-page.css` หน้าบทเรียนแยกพร้อมภาพและบทความเชิงกระบวนการสำหรับทุกหัวข้อย่อย
- `textbook.css` ระบบสี การจัดหน้า และ responsive layout
- `assets/lessons/` ภาพประกอบภูมิศาสตร์
