# ภูมิศาสตร์โลก: หนังสือดิจิทัล

หนังสือเรียนภูมิศาสตร์โลกภาษาไทยสำหรับผู้อ่านทั่วไป เรียบเรียงเนื้อหาจากกลไกและกระบวนการไปสู่รูปแบบเชิงพื้นที่ พร้อมภาพเต็มหน้า แผนภาพ ตาราง และสารบัญติดหน้าจอ

## เนื้อหา 6 หมวด

1. ธรณีสัณฐาน
2. ภูมิอากาศวิทยา
3. สมุทรศาสตร์
4. ภูมิภาคและลักษณะกายภาพโลก
5. การกระจายทรัพยากรธรรมชาติ
6. ภูมิศาสตร์อุตสาหกรรม

แต่ละบทมีภาพเปิดบท เนื้อหาแบบบทความ และหัวข้อย่อยรวม 96 เรื่อง ทุกหัวข้อเปิดเป็นหน้าบทเรียนแยกพร้อมแบบจำลองกระบวนการของตอน มาตราส่วนพื้นที่และเวลา เกณฑ์อ่านหลักฐาน ความสำคัญ และสรุปใจความ มีการนำทางไปเรื่องก่อนหน้าหรือเรื่องถัดไป รองรับเดสก์ท็อป แท็บเล็ต โทรศัพท์ โหมดกลางวันและกลางคืน

ทุกหัวข้อมี visual specification เฉพาะเรื่อง ระบุวัตถุประสงค์ สิ่งที่ต้องเห็น ป้ายไทย คำอธิบายในภาพ ข้อผิดพลาดที่ต้องหลีกเลี่ยง และสเกลพื้นที่–เวลา ภาพเชิงวิทยาศาสตร์ใช้แนวทางผสม: GPT Image สร้างฉากหรือหน้าตัด ส่วนแผนที่ ตัวเลข ลูกศร และข้อความที่ต้องตรงข้อมูลใช้ GIS หรือชั้นภาพที่ควบคุมได้ ภาพสำเร็จมีคำอธิบายในภาพและมีข้อความเดียวกันใน HTML เพื่อการเข้าถึง หัวข้อที่ยังไม่มีภาพเฉพาะเรื่องจะแสดงสถานะ “ภาพบริบทของหมวด” และไม่วนใช้ภาพอื่นเป็นหลักฐานโดยอัตโนมัติ

## เปิดใช้งาน

```powershell
node dev-server.mjs
```

จากนั้นเปิด `http://localhost:8080`

## ตรวจความสอดคล้องของภาพและบทเรียน

```powershell
npm run validate
```

ตัวตรวจสร้างไฟล์ข้อมูลสำหรับหน้าเว็บและยืนยันหัวข้อ 96 เรื่อง visual specification 96 รายการ แผนภาพตรงหัวข้อ process model 24 ตอน ไฟล์ภาพ ขนาด alt caption ข้อจำกัด และความสัมพันธ์ของ manifest ก่อนเผยแพร่

## ไฟล์สำคัญ

- `index.html` โครงหน้าหนังสือและส่วนแสดงผล
- `app.js` เนื้อหาทั้งหกหมวดและระบบเปลี่ยนบท
- `subtopic-content.js` รายละเอียดหัวข้อย่อยทั้ง 96 เรื่อง
- `content/visual-specs/` ข้อกำหนดภาพเฉพาะเรื่องครบ 96 หัวข้อ แบ่งเป็นสามกลุ่มเนื้อหา
- `topic-visual-specs.js` ไฟล์สำหรับหน้าเว็บที่สร้างจาก visual specification ด้วย `npm run build:specs`
- `visual-manifest.js` การจับคู่ภาพตามความหมาย สถานะภาพ คำกำกับ แหล่งข้อเท็จจริง และ process model 24 ตอน
- `scripts/validate-visual-manifest.mjs` ตัวตรวจความครบถ้วนของ visual manifest
- `scripts/validate-topic-visual-specs.mjs` ตัวตรวจ schema ชื่อหัวข้อ ความยาวคำอธิบาย และโหมดผลิตภาพ
- `docs/content-audit-*.md` ผลวิเคราะห์เนื้อหาและช่องว่างจากทีมธรณี น้ำ–มหาสมุทร และ human atlas
- `docs/next-batch-*.md` locked Thai copy, mustShow/mustAvoid, แหล่งอ้างอิง และลำดับผลิตชุดถัดไปที่ผ่าน cross-review
- `docs/atlas-production-roadmap.md` แผน P0–P2 และ publish gate สำหรับขยายจาก 96 หัวข้อเดิม
- `topic.html`, `topic-page.js`, `topic-page.css` หน้าบทเรียนแยกพร้อมภาพ ทฤษฎี และกระบวนการเกิดสำหรับทุกหัวข้อย่อย
- `textbook.css` ระบบสี การจัดหน้า และ responsive layout
- `assets/lessons/` ภาพประกอบภูมิศาสตร์
- `assets/ai/` ภาพ GPT Image แบบมีคำอธิบายไทยและไฟล์รุ่นที่เก็บประวัติการแก้ไข
- `assets/diagrams/` SVG ที่ควบคุมข้อความ สูตร ลูกศร topology และ legend ได้อย่างตรวจสอบซ้ำ
- `assets/lessons/PROMPTS.md` prompt set รุ่นสุดท้าย วิธีแก้เฉพาะจุด และเหตุผลเมื่อเปลี่ยนไปใช้ deterministic SVG
