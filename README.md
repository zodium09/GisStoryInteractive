# GeoStory Atlas

เว็บสารานุกรมภูมิศาสตร์เชิงเล่าเรื่องแบบ static สำหรับเปิดอ่านได้ทันทีจาก `index.html`

## สิ่งที่มีในหน้าเว็บ

- ความรู้พื้นฐานเรื่องภูมิศาสตร์ ภูมิประเทศ ภูมิอากาศ และการอ่านแผนที่
- แผนภาพกระบวนการ 9 ชุด พร้อมรูปภาพ ขั้นตอน และผลลัพธ์ของการเกิดภูมิประเทศ ภูมิอากาศ ภูเขาไฟ และปรากฏการณ์สำคัญ
- อภิธานศัพท์ภูมิศาสตร์ 82 คำสำหรับทบทวนคำสำคัญ
- การ์ดหัวข้อภูมิศาสตร์ 298 เรื่องพร้อมภาพ คำอธิบาย หมวดหมู่ และคำสำคัญ
- ปุ่มคำค้นเร็ว 81 คำสำหรับกรองเรื่องสำคัญ เช่น ภูเขาไฟ มรสุม ดินดอน ป่าฝน เมืองชายฝั่ง สะวันนา พื้นที่ชุ่มน้ำ อะทอลล์ เพอร์มาฟรอสต์ โอเอซิส ดินถล่ม ไฟป่า น้ำบาดาล น้ำกร่อย หินบะซอลต์ ทะเลเกลือ พายุฝุ่น ทะเลทรายชายฝั่ง โพลเดอร์ ทอร์นาโด น้ำขึ้นน้ำลง สเตปป์ ที่ราบน้ำพา ไต้ฝุ่น แนวปะการัง น้ำพุร้อน ทะเลปิด ITCZ ไจเออร์ ร่องลึก สันเขากลางมหาสมุทร น้ำฟอสซิล อัลบีโด เส้นหิมะ น้ำแข็งทะเล ลมคาตาบาติก เกาะความร้อนเมือง พื้นที่ถมทะเล โครงสร้างสีเขียว เกษตรขั้นบันได ป่าเมฆ พาราโม เงาฝน อีโคโทน น้ำท่วมทะเลสาบธารน้ำแข็ง ดินเหลว คลื่นพายุซัดฝั่ง คลื่นความร้อน ENSO เอลนีโญ ลานีญา MJO IOD แม่น้ำในบรรยากาศ น้ำใต้ดินเติมกลับ น้ำเค็มรุก hydrograph flood pulse และคานัต
- คำถามฝึกคิดแบบนักภูมิศาสตร์ 6 ข้อสำหรับใช้คู่กับแผนที่และบทความ
- ส่วนเปรียบเทียบแนวคิดที่มักสับสน เช่น ภูมิประเทศกับภูมิอากาศ อากาศกับภูมิอากาศ แม่น้ำกับลุ่มน้ำ
- เส้นทางสำรวจ 18 ชุดสำหรับอ่านหัวข้อที่เชื่อมกันเป็นเรื่องเดียว
- แผนผังจำแนกประเภทภูมิประเทศ ภูมิอากาศ แรงสร้างภูมิประเทศ และปัจจัยควบคุมภูมิอากาศ
- แถบสรุปจำนวนหัวข้อในแต่ละหมวดเพื่อช่วยอ่านเนื้อหาเยอะ ๆ ให้เป็นระบบ
- ค้นหาและกรองหัวข้อตามหมวด
- แผนที่ Leaflet/OpenStreetMap ที่เชื่อมกับหัวข้อแต่ละเรื่อง
- แผงอ่าน story telling พร้อมข้อมูลตำแหน่ง ขนาดพื้นที่ แนวคิดหลัก ภูมิอากาศ กล่อง info อ่านภาพ/กระบวนการภูมิศาสตร์-ธรณี กล่อง Academic Explainer ที่อธิบายนิยาม กระบวนการเกิด ประเภท ธรณีสัณฐาน วิธีอ่านภาพ/แผนที่ ประเด็นสำคัญ ลิงก์เปิดตำแหน่งจริง และตัวอย่างสถานที่เกี่ยวข้องบนแผนที่อย่างน้อย 6 รายการต่อเรื่อง

## การเปิดใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ หรือรันเว็บเซิร์ฟเวอร์เล็ก ๆ ในโฟลเดอร์นี้:

```powershell
node dev-server.mjs
```

แล้วเปิด `http://localhost:8080`
