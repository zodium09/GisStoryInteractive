# ภูมิศาสตร์กายภาพ

หนังสือเรียนบนเว็บฉบับอ่านต่อเนื่อง เรียบเรียงหัวข้อตามสารบัญหนังสือภูมิศาสตร์กายภาพที่ใช้อ้างอิง โดยลดศัพท์และเครื่องมือเชิงเทคนิคที่ไม่จำเป็น เนื้อหาอธิบายจากสาเหตุ ผ่านกระบวนการ ไปสู่ผลที่เกิดกับพื้นที่และชีวิตมนุษย์

## เนื้อหา

1. โลกและภูมิลักษณ์
2. ทรัพยากรธรรมชาติและการจัดการทรัพยากรธรรมชาติ
3. ภูมิอากาศและการเปลี่ยนแปลงภูมิอากาศ
4. ภูมิศาสตร์สิ่งแวดล้อม
5. ภัยธรรมชาติและการจัดการภัยพิบัติ

แต่ละบทเป็นบทความยาวหนึ่งเรื่อง มีหัวข้อย่อยตามลำดับเหตุผล ภาพประกอบ สรุปท้ายบท และปุ่มไปบทก่อนหน้าหรือบทถัดไป รองรับหน้าจอคอมพิวเตอร์และโทรศัพท์ โหมดกลางวันและกลางคืน รวมถึงการนำทางด้วยแป้นพิมพ์

## เปิดใช้งาน

```powershell
node dev-server.mjs
```

จากนั้นเปิด `http://localhost:8080`

## ไฟล์สำคัญ

- `index.html` โครงหน้าและส่วนแสดงผลหลัก
- `app.js` เนื้อหาทั้งห้าบทและการเปลี่ยนบท
- `textbook.css` รูปแบบหนังสือ การจัดหน้า และหน้าจอขนาดต่าง ๆ
- `assets/lessons/` ภาพประกอบกระบวนการ
