# แผนพัฒนา GeoStory Textbook Atlas

อัปเดต: 18 กรกฎาคม 2026

## เป้าหมาย

ทุกหัวข้อต้องมีคำอธิบายเฉพาะเรื่อง ภาพที่ตอบคำถามหลักของบท และความตรงกันสามทางระหว่างข้อเท็จจริง ภาพ และข้อความภาษาไทยในบทเรียน โดยเลือกวิธีผลิตให้เหมาะกับชนิดข้อมูล ไม่บังคับใช้ภาพ raster กับทุกสิ่ง

## สถานะฐานเนื้อหา

- หัวข้อเดิม 96 เรื่องมี summary และ visual specification ครบแล้ว
- visual specification แยก lesson objective, mustShow, mustAvoid, ป้ายไทย, มาตราส่วนพื้นที่–เวลา และ production mode
- หน้าเรื่องทั้ง 96 หน้าแสดงกรอบอ่านภาพและข้อจำกัดของภาพ
- ชุด P0 ที่ผลิตและตรวจแล้วครอบคลุมรอยต่อแผ่นธรณี โครงสร้างโลก การแยกชั้นของโลก งบน้ำ ชั้นหินอุ้มน้ำ การไหลเวียนมหาสมุทร การเกิดดิน ประเภททรัพยากร ธารน้ำแข็ง คาร์สต์ งบพลังงานโลก การกลับตัวของอุณหภูมิ และกำเนิด–การแตกตัวของคลื่น
- ชุด P1 เริ่มยกระดับการจำแนกภูมิลักษณ์ด้วยภาพการเคลื่อนตัวของมวลดินห้ารูปแบบที่แยก susceptibility ออกจาก trigger
- manifest รอบนี้มี 46 direct plans, 42 assets และ process models 24 ตอน; assets รุ่นใหม่ที่อนุมัติผ่าน cross-review ไม่ถูกนับรวมกับภาพเก่า 16 รายการที่ยังคงสถานะต้องตรวจโดยผู้เชี่ยวชาญ

## หลักเลือกวิธีผลิต

| สาระในภาพ | วิธีหลัก | เหตุผล |
|---|---|---|
| พื้นผิว ภูมิประเทศ ฉากตัดขวาง และภาพอธิบายที่ไม่ใช้เป็นข้อมูลวัด | GPT Image | ให้ภาพฐานแบบ atlas ที่เป็นธรรมชาติและช่วยมองระบบรวม |
| ป้ายไทย สูตร ลูกศร ขอบเขต ลำดับ และ topology | deterministic SVG/HTML | แก้ไข ตรวจข้อความ และทดสอบทิศทางได้ |
| พิกัด เส้นทาง เขตภัย ขอบเขตภูมิภาค และค่าปริมาณ | GIS/data | ต้องมีแหล่ง ช่วงเวลา หน่วย วิธีคำนวณ และ provenance |
| ภาพที่ต้องมีทั้งฉากและเรขาคณิตตรวจสอบได้ | hybrid | ใช้ GPT เป็น base-art และใช้ vector/data เป็นชั้นความหมาย |

## ลำดับผลิตต่อ

### P0 — prerequisite และ misconception สูง

1. ธรณี: โครงสร้างโลกสองเกณฑ์ → การเลื่อนทวีป → หลักฐานคลื่นไหวสะเทือนแบบ vector ray geometry
2. อุทก–สมุทรศาสตร์: เอลนีโญและลานีญา → น้ำขึ้นน้ำลง → ฝน การซึม และการเกิดน้ำไหลบ่า → ไฮโดรกราฟ → ~~กำเนิดและการแตกตัวของคลื่น~~
3. ภูมิภาค–ทรัพยากร: เกณฑ์แบ่งภูมิภาคกายภาพ → ห่วงโซ่อุปทานโลก → การกระจายทรัพยากรและความเสี่ยงจากการกระจุกตัว
4. บรรยากาศ: ~~องค์ประกอบ~~ → ~~โครงสร้างชั้น~~ → ~~งบพลังงาน~~ → ~~การกลับตัวของอุณหภูมิ~~ → เมฆ–หยาดน้ำฟ้า → มวลอากาศและแนวปะทะ

### P1 — ระบบและประเภทหลัก

1. ภูมิลักษณ์: ~~คาร์สต์~~ ~~มวลดิน~~ ที่ราบตะกอนน้ำพา แอ่ง ที่ราบสูง เกาะ และหมู่เกาะ
2. สมุทรศาสตร์: ไหล่ทวีป–ลาดทวีป ที่ราบก้นสมุทร สันเขากลางมหาสมุทร ร่องลึก–ภูเขาใต้ทะเล อุณหภูมิ ความเค็ม และความหนาแน่น
3. ทรัพยากร: แหล่งแร่ เชื้อเพลิงฟอสซิล พลังงานหมุนเวียน เหมือง การฟื้นฟู และเศรษฐกิจหมุนเวียน
4. อุตสาหกรรม: ปัจจัยทำเล การรวมกลุ่ม ท่าเรือ–ระเบียงเศรษฐกิจ เมืองอุตสาหกรรม เทคโนโลยี และบริการดิจิทัล

### P2 — หลักสูตรขยายหลัง 96 เรื่องเดิม

1. ทักษะ atlas และการรู้เท่าทันข้อมูลภูมิสารสนเทศ
2. ภูมิศาสตร์ประชากร เมือง และการย้ายถิ่น
3. Oceania, Arctic และ Antarctica
4. ป่า ความหลากหลายทางชีวภาพ และบริการระบบนิเวศ
5. ประมง พื้นที่ชุ่มน้ำ ทุ่งหญ้า และทรัพยากรร่วม
6. ระบบอาหาร–น้ำ–พลังงาน–ที่ดิน
7. สิทธิ การเข้าถึง และธรรมาภิบาลทรัพยากร
8. material footprint และทรัพยากรวิกฤต
9. ภูมิศาสตร์โครงสร้างพื้นฐานดิจิทัล
10. แรงงาน การเปลี่ยนผ่าน และความเป็นธรรมทางอุตสาหกรรม
11. ความเสี่ยง ภัย และความเปราะบางของมนุษย์

## Publish gate ต่อภาพ

1. ล็อก canonical claim และข้อความไทยในภาพ
2. แนบ authoritative source refs และระบุขอบเขตพื้นที่–เวลา
3. เลือก GPT raster, deterministic SVG, GIS หรือ hybrid ตามชนิดสาระ
4. ตรวจ mustShow/mustAvoid, ทิศลูกศร, topology, หน่วย และข้อจำกัด
5. ตรวจข้อความไทยด้วยการเทียบ exact copy และให้คำอธิบายเดียวกันอยู่ใน HTML
6. ให้ subagent ต่างสาขา cross-review และแก้ blocker ก่อนเชื่อมเป็น direct visual
7. ตรวจ XML/schema, manifest, alt text, responsive layout และเปิดครบ 96 หน้าโดยไม่มี page error หรือ horizontal overflow
8. บันทึก prompt, version, reviewer, review date และสถานะ QA โดยไม่เขียนทับรุ่นก่อน

## Definition of done

หัวข้อหนึ่งถือว่าเสร็จเมื่อมี direct visual ที่ผ่าน publish gate, ไม่มีภาพบริบทถูกใช้แทนหลักฐาน, ข้อความและภาพไม่ขัดกัน, มีข้อจำกัดและแหล่งอ้างอิง, และเปิดอ่านได้ทั้งเดสก์ท็อปกับมือถือ ส่วนหัวข้อที่ยังไม่ผ่านให้คงสถานะภาพบริบทอย่างตรงไปตรงมา ไม่เลื่อนเป็น approved เพื่อเพิ่มตัวเลขความครอบคลุม
